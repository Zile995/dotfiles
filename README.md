# dotfiles

## Necessary Arch Linux packages for zsh:
  * ```Shell
    sudo pacman -S fzf fd exa bat git-delta zsh-completions
    ```
  * Install the clipboard command line tools: `wl-clipboard` for Wayland or `xsel` for X11
